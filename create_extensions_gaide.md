# Руководство по созданию расширений для Visibility Browser (.visibility)

Ваш браузер `Visibility Browser` поддерживает установку собственных расширений, написанных на языке JavaScript. 

Расширение представляет собой файл конфигурации в формате **JSON**, просто переименованный в `.visibility`.
Вы можете установить расширение, нажав на кнопку с иконкой пазла **🧩** в верхней панели браузера!

---

## 🚀 Как написать свое первое расширение?

1. Создайте обычный текстовый файл.
2. Вставьте в него следующий базовый шаблон:

```json
{
  "manifest": {
    "name": "Мое Супер Расширение",
    "version": "1.0",
    "author": "Ваш Никнейм",
    "description": "Меняет фон всех сайтов и добавляет кнопку"
  },
  "ui_script": "visib.addButton('🪄', 'Сделать магию', () => { visib.executeInActiveTab('alert(\"Магия!\")'); });",
  "ui_style": "#tabs-bar { background: #222 !important; }",
  "ui_html": "<div style='position:fixed; bottom:20px; right:20px; color:white; background:rgba(0,0,0,0.8); z-index:999999; padding:10px; border-radius:8px;'>Привет!</div>",
  "content_script": "document.body.style.filter = 'grayscale(50%)';"
}
```

3. Сохраните этот файл под названием `my_extension.visibility`.
4. Откройте **Visibility Browser**, нажмите на **🧩** и выберите этот файл!
5. Перезапустите браузер (закройте и откройте заново).

## 🛠 Описание полей (API)

В файле `.visibility` есть несколько главных разделов:

### 1. `"manifest"`
Здесь указывается техническая информация (название, версия, описание). 
- **`icon`**: Ссылка (URL) или Base64 на иконку расширения. Она будет отображаться в списке установленных расширений.

### 2. `"ui_script"`
Это JavaScript код, который исполняется **в интерфейсе самого браузера**.
Доступные методы объекта **`visib`**:

- **`visib.addButton(иконка, подсказка, функция_клик)`**  
  Добавляет кнопку в трей расширений (внутри 🧩).  
  *Иконка*: Эмодзи (`'⭐'`) или ссылка (`'https://...'`).

- **`visib.showPopup(html, callback)`**  
  Открывает модальное окно с вашим HTML. В `callback(box)` придет элемент окна.

- **`visib.tabs`** (Управление вкладками)  
  - `visib.tabs.create(url)` — Новая вкладка.
  - `visib.tabs.close(id)` — Закрыть вкладку.
  - `visib.tabs.switch(id)` — Перейти на вкладку.
  - `visib.tabs.getAll()` — Список всех вкладок.
  - `visib.tabs.getActiveId()` — ID текущей вкладки.
  - `visib.tabs.duplicate(id)` — Дублировать.
  - `visib.tabs.executeInAll(code)` — Выполнить код во всех вкладках сразу.

- **`visib.window`** (Управление окном браузера)  
  - `visib.window.minimize()` / `maximize()` / `close()` — Управление окном.
  - `visib.window.setFullScreen(true/false)` — Полноэкранный режим.
  - `visib.window.isMaximized(cb)` — Проверить, развернуто ли окно.

- **`visib.navigation`** (Навигация в активной вкладке)  
  - `visib.navigation.back()` / `forward()` / `reload()` / `stop()`
  - `visib.navigation.goTo(url)` — Перейти по адресу.

- **`visib.page`** (Взаимодействие со страницей)  
  - `visib.page.executeJS(code)` — Выполнить JavaScript на сайте.
  - `visib.page.insertCSS(css)` — Внедрить свои стили на сайт.
  - `visib.page.capturePage(cb)` — Сделать скриншот страницы (Base64 в cb).

- **`visib.cookies`** (Работа с данными сайтов)  
  - `await visib.cookies.get(filter)` — Получить куки.
  - `await visib.cookies.set(details)` — Установить куки.
  - `await visib.cookies.remove(url, name)` — Удалить.
  - `visib.cookies.clearAll()` — Очистить всё.

- **`visib.vpn`** (Управление VPN 🛡️)  
  - `await visib.vpn.getStatus()`, `connect(idx)`, `disconnect()`, `addLink(link)`, `fetchSub(url)`.

- **`visib.app`** (Системное)  
  - `visib.app.getVersion()` — Версия браузера.
  - `visib.app.relaunch()` — Перезапустить браузер.

- **`visib.notifications.show(msg)`** — Показать уведомление (toast).

- **`visib.setValue(key, val)`** / **`visib.getValue(key)`** — Постоянная память расширения.

*(Важно: Код в `ui_script` пишется в 1 строку, не используйте переносы строк, так как это JSON формат. Либо используйте `\n` )*

### 3. `"ui_style"`
Это ваш кастомный CSS-код. Он позволяет полностью менять внешний вид (дизайн) самого браузера. Вы можете перекрашивать кнопки, изменять форму вкладок, скрывать стандартные элементы интерфейса и делать любые визуальные модификации (создать свой уникальный и неповторимый вид). Все стили применяются мгновенно при загрузке расширения!

### 4. `"ui_html"`
Позволяет вам интегрировать собственный HTML-код прямо в структуру браузера. Это удобно для добавления собственных виджетов, выплывающих панелей или других элементов, с которыми потом можно взаимодействовать через `ui_script`.

### 5. `"content_script"`
Это JavaScript код, который **автоматически внедряется на вообще все сайты**, которые вы открываете в браузере (аналог Tampermonkey или Content-скриптов в Chrome).

- Он выполняется в тот момент, когда сайт(вкладка) сообщает, что "Загрузка завершена" (`dom-ready`).
- Здесь вы можете менять цвета сайта, удалять рекламные блоки через `document.querySelectorAll('.ads').forEach(el => el.remove())`, читать куки и делать что угодно с DOM.

## ⚠️ Частые вопросы и ошибки

**1. "Ошибка файла расширения (Неверный формат)"**
Убедитесь, что ваш `.visibility` файл является строгим, правильным JSON. 
Значения должны быть в кавычках `""`. 
Одинарные кавычки в коде нужно прописывать внутри двойных.
Внутри кода нельзя использовать переносы строк физически по кнопке `Enter`, если нужно – пишите все в одну строчку файла.

**2. Можно ли сделать как AdBlock?**
Да. Вы можете прописать в `"content_script"` логику, которая будет находить элементы по `id` или `class` и задавать им `display: none;`.

**3. Где лежат установленные расширения?**
На Windows они автоматически копируются по пути: `C:\Users\ВашеИмя\AppData\Roaming\visibility-browser\extensions\`
Вы можете зайти туда и редактировать файлы расширений "наживую", чтобы проверить, как работает код, просто перезагружая ваш браузер.
